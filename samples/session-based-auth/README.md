## Description
本教程是基于`express-session`，`passport`，`@nestjs/passport`，`redis`实现的授权登录。

[基于session的授权登录实现教程](https://www.loginradius.com/blog/engineering/guest-post/session-authentication-with-nestjs-and-mongodb/) 
## 说明

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
